## 🌿 Objectif du Projet

**ArtistFloral** est un site vitrine moderne conçu pour mettre en avant le savoir-faire d'un artisan fleuriste. L'objectif est d'offrir une expérience utilisateur fluide et immersive, mettant l'accent sur la beauté des visuels et la clarté des services proposés.

---

## 🛠 Stack Technique

Ce projet utilise des outils modernes pour un rendu rapide et un design soigné :

- **HTML5** : Structure sémantique.
- **Tailwind CSS** : Framework utilitaire pour un design sur mesure et responsive sans quitter le HTML.
- **JavaScript (ES6+)** : Pour les animations, la gestion du menu mobile et l'interactivité.

---

## ✨ Points Forts

- **Design Minimaliste :** Une interface épurée qui laisse place aux photos de compositions florales.
- **Performance :** Optimisation du CSS grâce au JIT (Just-In-Time) de Tailwind.
- **Responsive :** Entièrement adapté aux smartphones et tablettes.
- **Typographie :** Choix de polices élégantes pour refléter l'aspect artisanal.

---

## 🚀 Installation & Développement local

1.  **Cloner le projet :**

    ```bash
    git clone [https://github.com/DeVeo44/ArtistFloral.git](https://github.com/DeVeo44/ArtistFloral.git)
    ```

2.  **Installer les dépendances (si tu utilises npm pour Tailwind) :**

    ```bash
    npm install
    ```

3.  **Lancer le build Tailwind en mode "watch" :**
    ```bash
    npm run watch
    ```

---

## 📂 Structure du code

- `/src` : Contient les fichiers source HTML et le CSS brut.
- `/dist` (ou `public`) : Contient les fichiers finaux optimisés.
- `tailwind.config.js` : Configuration personnalisée des couleurs et thèmes.
